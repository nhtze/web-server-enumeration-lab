# Web Server Enumeration Lab

## Objective
Build a web server laboratory using Ubuntu Server and perform reconnaisance and enumeration
against an Apache HTTP service using Nmap and NSE scripts.


## Project Overview
The objective of this project was to understand how attackers gather publicly available information from web servers before attempting exploitation.
The assessment focused on identifying exposed services, enumerating HTTP configuration, analyzing HTTP headers, identifying supported HTTP methods, 
and evaluating whether the collected information represented an actual security risk.
